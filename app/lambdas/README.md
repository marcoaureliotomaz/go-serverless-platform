# Lambdas

Cada subpasta representa uma Lambda da POC.

## Papel de cada Lambda

- adaptar o evento HTTP recebido
- acionar o caso de uso correspondente
- devolver a resposta HTTP padronizada
- propagar logs, metricas e traces
